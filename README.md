# ซูโดกุ (Dad Sudoku)

เกมซูโดกุเล่นคนเดียว กรอกเลขเอง ตาราง 9x9 ทำด้วย Vue 3 + Vite เป็น PWA
ติดตั้งเป็นแอปบนมือถือได้โดยไม่ต้องผ่าน App Store / Play Store

## ฟีเจอร์

- **สร้างโจทย์เอง** — กรอกเลขตั้งต้นทั้งหมดด้วยตัวเอง กด "ยืนยันโจทย์" แล้วระบบจะเช็คให้อัตโนมัติว่า:
  - มีเลขซ้ำกันในแถว/คอลัมน์/บล็อกไหม (ขึ้นข้อความเตือน)
  - โจทย์แก้ได้จริงไหม (ไม่งั้นขึ้นว่าแก้ไม่ได้)
  - คำตอบมีแบบเดียวหรือหลายแบบ (ต้องเหลือคำตอบเดียวถึงจะเริ่มเล่นได้)
  - ผ่านหมดแล้วจะล็อกเลขที่กรอกเป็นโจทย์ แล้วเข้าสู่โหมดเล่นปกติทันที
- สุ่มโจทย์ใหม่ทุกครั้ง รับประกันคำตอบเดียว (unique solution) 4 ระดับ: ง่าย / ปานกลาง / ยาก / โหด
- โหมดจดเลข (pencil marks) แยกจากโหมดกรอกคำตอบจริง
- ปุ่ม "ช่วยจด" เติม candidate ที่เป็นไปได้ให้อัตโนมัติ
- ไฮไลต์แถว/คอลัมน์/บล็อกที่เกี่ยวข้อง และช่องที่ค่าเลขซ้ำกัน (ตัดกฎ)
- Undo / Redo
- จับเวลา
- เซฟเกมอัตโนมัติลง localStorage — ปิดแอปแล้วกลับมาเล่นต่อได้
- ปุ่มตัวใหญ่ ตัวหนังสือคมชัด contrast สูง เหมาะกับผู้สูงวัย

## วิธีรันดูก่อน (บนคอมพิวเตอร์)

```bash
npm install
npm run dev
```

เปิด `http://localhost:5173` ในเบราว์เซอร์

## วิธี deploy ให้พ่อเล่นบนมือถือจริง (แนะนำ: Vercel)

1. สมัคร/ล็อกอิน vercel.com ด้วย GitHub
2. push โปรเจกต์นี้ขึ้น GitHub repo (หรือใช้ `vercel` CLI จาก terminal ตรงๆ ก็ได้โดยไม่ต้องมี repo)
3. ที่หน้า Vercel กด **Add New Project** → เลือก repo นี้ → กด Deploy (ไม่ต้องตั้งค่าอะไรเพิ่ม Vercel จะจับ Vite ให้เอง)
4. จะได้ลิงก์ประมาณ `https://xxxx.vercel.app`
5. ส่งลิงก์นั้นให้พ่อเปิดใน **Chrome บนมือถือ**
6. กดเมนู (จุดสามจุดมุมขวาบน) → **"เพิ่มไปยังหน้าจอหลัก" / "ติดตั้งแอป"**
7. จะได้ไอคอนแอปบนหน้าจอหลัก เปิดแล้วเต็มจอเหมือนแอปจริง ไม่มีแถบ URL

หรือถ้าไม่อยากใช้ GitHub เลย รันตรงจากเครื่อง:

```bash
npm install -g vercel
npm run build
vercel deploy --prod
```

deploy ใหม่ทุกครั้งที่แก้โค้ด แค่รันคำสั่งเดิมซ้ำ พ่อไม่ต้องโหลดอะไรใหม่เอง
(service worker จะอัปเดตให้อัตโนมัติตอนเปิดแอปครั้งถัดไป)

## โครงสร้างโปรเจกต์

```
src/
  utils/sudokuEngine.js   // generator + solver (unique-solution digger)
  composables/useGame.js  // state, undo/redo, localStorage, timer
  components/
    SudokuGrid.vue        // ตาราง 9x9
    NumberPad.vue         // ปุ่มกดเลข + โน้ต/ลบ
    TopBar.vue            // เวลา, undo/redo, เกมใหม่
    WinModal.vue          // ป๊อปอัปตอนไขสำเร็จ
  App.vue                 // เชื่อมทุกอย่างเข้าด้วยกัน + หน้าเลือกระดับ
```

## ปรับแต่งง่ายๆ ที่อาจอยากทำต่อ

- เปลี่ยนจำนวน clue ต่อระดับ: แก้ `DIFFICULTY_CLUES` ใน `sudokuEngine.js`
- เปลี่ยนสี/ธีม: แก้ CSS variables ใน `src/style.css` (ตัวแปร `--paper`, `--indigo`, `--gold` ฯลฯ)
- ปิดโหมดจดเลขไปเลยถ้าไม่อยากให้ซับซ้อน: ซ่อนปุ่ม "จดโน้ต" กับ "ช่วยจด" ใน `NumberPad.vue`
